<introduction>

Diet Optimizer

Twitch Annotator is my project for CMSC 190 entitled "Special Problem". The aim of the project is to create a google chrome extension that could classify the emotions of the chat messages found in a Twitch.tv recorded live stream.


The technologies used in this project are NodeJS, Twitch API, and Google Chrome extension.

</introduction>


<methodology>

The google chrome extension used artificial intelligence in a form of supervised machine learning such as Naive Bayes Classifier and Support Vector Machines to categorize the chat messages into their correct sentiment.


The sentiment was then used to annotate which parts of the recorded live stream are amusing, infuriating, pathetic, or neutral. The google chrome extension showed this by using a specific color for each sentiment and presented the aggregation of colors below the livestream

</methodology>
