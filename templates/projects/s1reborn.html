<introduction>

SystemOne: Reborn

SystemOne is an online enrollment system in the University of the Philippines Los Baños (UPLB)

made by the UPLB students. It was regarded as a slow website where the student needs to wait at

least 5 seconds so that a page will load. There are even cases that the page will not load at all.

Fortunately, the latest version of SystemOne, codenamed 'Reborn', fixed all of the issues faced

by the previous versions of SystemOne.

</introduction>

<methodology>

We used NodeJS on the server side with MySQL as the database of the website.

The design uses Materialize CSS Framework and jQuery.

Amazon Web Service is used to provide the server needed to deploy the website.

</methodology>

<contribution>

I developed the automated testing of the back-end routes of the website.

I also managed the website by answering the questions of the students

who have a hard time using the website.

</contribution>
