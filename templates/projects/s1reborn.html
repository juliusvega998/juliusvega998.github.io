<introduction>

SystemOne: Reborn

SystemOne is an online enrollment system in the University of the Philippines

Los Baños (UPLB) made by the students themselves. It was regarded as a

slow website where the student needs to wait at least 5 seconds so that a web page

will load. There are even cases that the page will not load at all.

Fortunately, the latest version of SystemOne, codenamed 'Reborn', fixed

all of the issues faced by the previous versions of SystemOne.

</introduction>

<methodology>

We used NodeJS on the server side with MySQL as the database of the website.

The design uses Materialize CSS Framework and jQuery.

Amazon Web Service is used to provide the server needed to deploy the website.

</methodology>

<contribution>

I developed the scripts needed for automated testing of the back-end routes.

I also managed the website by answering the questions of the students

who had a difficulty while using the website.

</contribution>
