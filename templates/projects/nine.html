<introduction>

Project NINE: UP CIEM Website

Project NINE is a fund raising project of Alliance of Computer Science Students (ACSS)

which aims to renovate the website of UP Circle of Industrial Engineer Majors (CIEM).

UP CIEM is an Academic Organization based in UP Diliman. The website can be found here.

</introduction>

<methodology>

We used native PHP server side with Joomla! as a Content Management System.

The design of the website uses Materialize CSS Framework

as well as jQuery to add scripts on the website.

CPanel is used to deploy the website over the internet.

</methodology>
