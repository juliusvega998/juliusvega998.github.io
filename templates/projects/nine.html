<introduction>

Project NINE: UP CIEM Website

Project NINE is a fund raising project of Alliance of Computer Science Students

which aims to improve the website of UP Circle of Industrial Engineer Majors.

UP CIEM is an Academic Organization based in UP Diliman.

</introduction>

<methodology>

We used native PHP server side with Joomla! as a Content Management System.

The design of the website uses Materialize CSS Framework

as well as jQuery to add scripts on the website.

CPanel is used to deploy the website over the internet.

</methodology>

<contribution>

I created the scripts for the events page in the IEshikawa subdomain page.

I also implmemented the design of the mobile view of the home page.

</contribution>
